FileZilla.

FileZilla es un software de transferencia de archivos sobre el protocolo ftp, permite a los usuarios la descarga de ficheros desde el servidor. Los usuarios deben usar un software cliente de FileZilla, para conectarse a un servidor FileZilla, una vez conectados pueden copiar a su ordenador el contenido del servidor.
